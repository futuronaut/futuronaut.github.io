# The Closed Loop Theory of Consciousness

This is a summary of some of my ideas about consciousness. As I tried to explain these ideas to others I realized it’s 
not easy to convey these thoughts and the basic idea might be too subtle to be easily understood. Now I decided to try 
again and approach it like an open source software project: release early, release often. So please forgive me the 
roughness of this text, it’s a work in progress.
